\section{Multiple Query Optimization}

A Query\cite{codd_relational_1970} is a demand for information to be pulled from a database. A query can vary in size and structure, as well as in execution time. These are written with \code{SQL}, which is often specialized to the corresponding hardware\cite{shirgoldbird_microsoft_nodate}\cite{} it runs on.
